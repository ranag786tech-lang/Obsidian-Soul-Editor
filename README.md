## 🛠️ Quick Start

### Prerequisites
- Node.js 18+
- npm or yarn
- A soul ready to create ✨

### Installation

```bash
# Clone the sanctuary
git clone https://github.com/yourusername/obsidian-soul-editor.git

# Enter the sacred space
cd obsidian-soul-editor

# Install dependencies
npm install

# Awaken the editor
npm start
```
